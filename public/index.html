🔧 Webpack Loader Runner Worker Demo

使用 Web Worker 运行 webpack loaders，体验非阻塞的文件处理

💡 什么是 Webpack Loader Runner Worker？

  • 将 webpack loaders 处理放在 Web Worker 线程中执行
  • 避免 loader 处理阻塞主线程的 UI 渲染和用户交互
  • 支持 TypeScript、Babel、CSS 等各种 loader 组合
  • 通过消息传递机制与主线程通信，返回处理结果
